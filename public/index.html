X-Ray Goggles

Remix Any Page on the Web

X-Ray Goggles makes it easy to see and remix the code behind your favorite web pages. The best part: you don't need any prior coding experience! Just activate Goggles, click on a section, and edit. Then, share your remixes with others.

How to get started:

  1. Open Your Bookmark Toolbar

    If you're using Chrome, click the options icon to the right of your URL bar, select Bookmarks, then make sure Show bookmarks bar is selected.

    Instructions for other platforms:

    • Windows: 
      • Chrome
      • Firefox
      • Internet Explorer
    • Mac: 
      • Chrome
      • Firefox
      • Safari
    • Other: 
      • Chrome
      • Firefox

  2. Install Goggles

    Click and drag this button to your bookmark toolbar to install X-Ray Goggles.

About X-Ray Goggles

X-Ray Goggles was developed by Mozilla. The original version of the project that ended in 2019 included the ability to save remixes.

This version was created by Mouse using the original Mozilla version. This version does not support saving web page remixes, but we recommend taking a screenshot of the remixed page to show your work.

Mouse uses this tool in our curriculum platform Mouse Create. To learn more about Mouse's Web Literacy curriculum, and other curriculum visit Mouse Create!.
